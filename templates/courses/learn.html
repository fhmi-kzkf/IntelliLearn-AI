Key Concept: The Artificial Neuron

An artificial neuron, also called a perceptron, is the basic unit of a neural network. It receives inputs, applies weights to them, and produces an output through an activation function.

# Simple neuron calculation
inputs = [0.5, 0.3, 0.8]
weights = [0.2, 0.8, -0.5]
bias = 0.1

# Weighted sum
weighted_sum = sum(i * w for i, w in zip(inputs, weights)) + bias

# Apply activation function (sigmoid)
output = 1 / (1 + exp(-weighted_sum))
